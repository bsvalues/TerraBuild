### 3. **Auto-Port Handling & Retry Logic**

Include logic like:

import socket
def find_free_port(start=8000):
    port = start
    while port < 9000:
        with socket.socket(socket.AF_INET, socket.SOCK_STREAM) as s:
            try:
                s.bind(("", port))
                return port
            except OSError:
                port += 1
